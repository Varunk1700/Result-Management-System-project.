# Student Result Management System

## Description
The Student Result Management System is a desktop application developed using **Python**, **Tkinter**, and **SQLite**. This project is designed to manage student information, course details, and examination results through a simple and user-friendly graphical interface. It allows users to add and manage student records, create and update course information, enter marks for students, and view result reports by searching with a roll number. All data is stored in an SQLite database, ensuring that records are saved permanently and can be accessed anytime.

## Features
- Add and manage student details  
- Add and manage course information  
- Enter student marks and results  
- View result reports using roll number  
- Data stored securely using SQLite  
- Simple and easy-to-use GUI  

## Technologies Used
- Python  
- Tkinter  
- SQLite3  
- Pillow (PIL)
